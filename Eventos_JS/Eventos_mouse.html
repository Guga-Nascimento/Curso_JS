Mouse

  • onclick

    • Acionado quando o elemento é clicado

  • ondblclick

    • Acionado quando o elemento é clicado duas vezes

  • onmouseup

    • Acionado quando o click sobre o elemento determinado é liberado

  • onmouseover

    • Acionado quando o cursor do mouse sobre-põe o elemento

  • onmouseout

    • Acionado quando o cursor do mouse sai da região da página oculpada pelo o elemento
